# Tai Lieu Tham Chieu Dang Co

> Danh sach nay giup AI Agent biet nen tim nguon nao khi can hieu de bai, mau bao cao, mau slide va cac ghi chu hoc phan.

## Nhom file de bai va mau nop bai

| File | Vai tro tham chieu |
|---|---|
| `docs/1. Assignment .docx` | Nguon goc ve de bai, san pham phai nop, tieu chi cham diem va cac yeu cau chinh thuc. Khi co tranh cai, doc file nay truoc. |
| `docs/2. Mau tai lieu.docx` | Mau whitepaper/bao cao Word. Dung de canh bo cuc, trang bia, cac chuong va cach trinh bay. |
| `docs/3. Mau bao cao.pptx` | Mau slide bao cao. Dung khi lam slide thuyet trinh/bao ve. |
| `docs/Ten-de-tai.txt` | Ten de tai da chot/ghi chu ten de tai. |

## Ghi chu khi dung file Word/PPT

- File goc nen duoc giu nguyen de doi chieu.
- Khi can cho AI Agent doc on dinh hon, nen chuyen noi dung quan trong sang Markdown trong thu muc nay.
- Khong nen copy nguyen tat ca noi dung Word vao mot file duy nhat; nen tach thanh tom tat theo chu de: de bai, bao cao, notebook, database, AI coworking.
