# plans/ — Nơi lưu các file kế hoạch làm việc

Thư mục này chứa các **file kế hoạch (plan)** của nhóm khi triển khai từng phần việc của dự án —
ví dụ kế hoạch làm app Streamlit, viết whitepaper, làm slide, notebook 07... Mỗi việc một file riêng.

Mục đích: trước khi bắt tay vào một phần việc lớn, viết ra kế hoạch (mục tiêu, các bước, việc cần
làm, bẫy cần tránh) để cả nhóm cùng thấy và bám theo — thay vì giữ trong đầu hoặc chỗ riêng lẻ.

## Quy ước

- **Định dạng:** file Markdown (`.md`).
- **Đặt tên:** `plan-<tên-việc>.md`, viết thường không dấu, nối bằng gạch ngang.
  Ví dụ: `plan-app-streamlit.md`, `plan-whitepaper.md`, `plan-slide.md`, `plan-nb07.md`.
- **Được commit chung:** các file trong đây **được đưa lên GitHub** để cả nhóm cùng đọc, không phải
  file nháp cá nhân. Ghi chú/tiến độ riêng của mỗi người vẫn để ở `context/<tên>.md` như cũ.
